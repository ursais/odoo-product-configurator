==============================
Product Configurator Duplicate
==============================

This module will allow to duplicate product template along with:
* attribute, attribute values
* configuration steps
* configuration restrictions, 

Usage
=====

To use this module:

* Go to Sales > Configurable Products > Action > Duplicate
